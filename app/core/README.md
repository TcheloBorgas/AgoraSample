# app/core/

## Português (Brasil)
- `config.py`: classe `Settings` com variáveis de ambiente (`.env`).
- `database.py`: conexão singleton com MongoDB (`get_db()`).
- `logging_config.py`: logging estruturado em JSON.
- `metrics.py`: contadores e tempos médios em memória.

Tecnologias: `pydantic-settings`, `pymongo`, `logging`.

---
## English
Core infrastructure: settings, database, structured logs, and in-memory metrics.
