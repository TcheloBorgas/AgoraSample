# app/schemas/

## Português (Brasil)
- `api.py`: contratos HTTP (requests/responses), incluindo `AssistantResponse`.
- `agent_trace.py`: estrutura do trace operacional por turno.
- `proactive_suggestion.py`: estrutura de sugestões proativas.

Esses schemas garantem consistência dos dados entre backend e frontend.
